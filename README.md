# ShortcutCovid-19

日本全体と和歌山県内の新型コロナウイルス(COVID-19)感染者数をサクッと確認するショートカットのためのPythonスクリプト

## Requirement
- iOS/iPadOS Device
- Shortcut App
- Pythonista(iOS/iPadOS App)
